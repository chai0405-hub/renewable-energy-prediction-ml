# Renewable Energy Output Prediction System using Machine Learning

This project predicts renewable energy adoption using a Machine Learning model and Flask web framework.

## Features
- Decision Tree Classifier
- Flask REST API
- Web UI using HTML
- Local deployment

## Project Structure
- app.py : Flask application
- train_model.py : Model training
- model.pkl : Trained model
- data/ : Dataset
- templates/ : HTML UI

## How to Run

```bash
pip install -r requirements.txt
python train_model.py
python app.py
